# Home Fix Solution

Production-ready Next.js website for Home Fix Solution with garage door services prioritized, plus air duct cleaning and chimney cleaning pages.

## Run Locally

```bash
npm install
npm run dev
```

## Notes

- Form routes are backend-ready placeholders under `/api/book`, `/api/contact`, and `/api/careers`.
- Replaceable business details are styled in blue with the `replaceable` class.
- Sitemap, robots, favicon, metadata, Open Graph, Twitter cards, canonical URLs, breadcrumbs, and schema markup are included.
